# Welcome to xiaozhi-mcphub

xiaozhi-mcphub is an enhanced MCP (Model Context Protocol) tool bridge specifically optimized for Xiaozhi AI platform. Built upon the excellent MCPHub foundation, it adds intelligent tool synchronization, automatic reconnection, and seamless AI platform integration.

## Quick Start

Get xiaozhi-mcphub running in minutes with Docker:

```bash
docker run -p 3000:3000 huangjunsen0406/xiaozhi-mcphub
```

Or with custom configuration:

```bash
docker run -p 3000:3000 -v $(pwd)/mcp_settings.json:/app/mcp_settings.json huangjunsen0406/xiaozhi-mcphub
```

Access the dashboard at `http://localhost:3000` with default credentials:

- Username: `admin`
- Password: `admin123`

## Core Concepts

### MCP Endpoints

xiaozhi-mcphub provides multiple ways to access your MCP servers:

- **Unified Access**: `http://localhost:3000/mcp` - Access all servers
- **Group Access**: `http://localhost:3000/mcp/{group}` - Access specific groups
- **Server Access**: `http://localhost:3000/mcp/{server}` - Access individual servers
- **Smart Routing**: `http://localhost:3000/mcp/$smart` - AI-powered tool discovery

### Protocol Support

- **HTTP MCP**: Modern streamable HTTP interface (recommended)
- **SSE**: Server-Sent Events for legacy compatibility
- **stdio**: Native MCP protocol for server communication
- **WebSocket**: Xiaozhi AI platform native connection

## Xiaozhi AI Integration

xiaozhi-mcphub provides enhanced features specifically for Xiaozhi AI platform:

- **Automatic Tool Sync**: Tools are automatically synchronized when enabled/disabled
- **Smart Reconnection**: Intelligent reconnection ensures Xiaozhi gets the latest tool list
- **Real-time Updates**: Tool status changes are instantly reflected in Xiaozhi
- **Optimized Performance**: Reduced token consumption through intelligent tool filtering
